# Library-Managment-System

**Library Management System** developed using **Python** and **SQL** for managing books, members, and borrowing transactions.

This project demonstrates basic database usage and Python-based data operations.

---

## 📌 Project Overview

This project implements a basic library management system that allows:

- Managing books
- Viewing stored records
- Borrowing books
- Returning books
- Managing members
- Managing study room

The system uses a relational database and Python scripts to perform data operations.

---

## ⚙️ Technologies Used

- Python
- SQL
- SQLite
- QT Designer
